Protocolo TCP/IP

Definicion

Es un conjunto de normas y procedimientos útiles para la transmisión de datos, conocido por el emisor y el transmisor. Se trata de un sistema abierto en la medida en que la definición de la serie de protocolos, Así como muchas de sus implantaciones, se encuentran disponibles al público. Ello constituye la base para lo que se conoce como la red Internet mundial, con millones de computadoras interconectadas (Crespo & Candelas, 1998) El nombre de este protocolo está compuesto por 2 de sus protocolos más importantes. TCP (Transmission Control Protocol) y el protocolo IP (Internet Protocol). Este posibilita que los dispositivos conectados a internet se comuniquen entre sí en varias redes.
